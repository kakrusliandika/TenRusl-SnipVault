How to use SnipVault

Quick start, tips, offline, backup—everything in one page.

Quick Start

  1. Click New to create a snippet.
  2. Fill Title, choose Language, type your code/notes.
  3. Add Tags (comma separated) for easy filtering.
  4. Mark Favorite or Pinned if needed.
  5. Press Save (or Ctrl+S).

Capture & Edit

  • Duplicate from an existing snippet using Duplicate.
  • Use Notes panel (markdown-lite) for explanations.
  • Click Copy to copy code to clipboard.

Search & Filter

  • Global search supports title, content, and tags.
  • Filter by Language, Tags, Favorites, Pinned.
  • Toggle AND mode to match all tags.
  • Sort by updated_at, created_at, title, or language.

Import & Export

  • Click Export to download .json of all snippets.
  • Use Import to load a previous export; conflicts are auto-checked.
  • Choose strategy: skip, overwrite-id, or always-insert (see preview).

Backup & Restore

  • Enable periodic backup reminders in Settings (e.g., every 7 days).
  • Keep exports in cloud storage or version control.

Keyboard Shortcuts

  • Ctrl+K — Global search
  • Ctrl+S — Save
  • N — New snippet
  • Del — Delete (with confirm)

Offline & PWA

  • Install as App (PWA) for 1-click access; works fully offline.
  • Updates are small; if a banner appears, reload to apply.

Troubleshooting

  • If list is empty, check filters and clear search.
  • If save fails, ensure browser storage is not blocked (incognito quotas).
  • Import errors: confirm file is a valid SnipVault export.

Privacy & Security

  • Data stays in your browser (IndexedDB). Optional local encryption via Secure Mode.
  • Always export before clearing site data or reinstalling the browser.
